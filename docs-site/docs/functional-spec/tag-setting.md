# タグ設定

```gherkin
Feature: タグ設定
  As a ログイン中のユーザー
  I want タグの選択を変更したい
  So that 興味のある分野のフィードを調整できる

  Scenario: 画面に遷移したとき
    When 画面に遷移してくると
    Then GitHubでアクティブな順番に、タグが20件単位でページネーションされて表示される
      And タグの並び順は、おおよその人気順
      And 既に選択されているタグがアクティブな状態になっている
    # https://madnight.github.io/githut/#/pull_requests/2024/1
    # 上記から、20件でメジャーな言語を抑えられると考えられる
    # GitHubのPRの数の順番等にしたいが、20件単位なのである程度雑で良いと思われる

  Scenario: ページネーション - 次のページ
    When Nextボタンを押すと
    Then タグの次の20件に表示が切り替わる

  Scenario: ページネーション - 前のページ
    When Backボタンを押すと
    Then タグの前の20件に表示が切り替わる

  Scenario: タグを検索する
    When 検索ボックスにキーワードを入力すると
    Then タグ名に部分一致するタグが人気順で表示される
      And 検索結果は20件単位でページネーションされる
      And ページ番号は1にリセットされる

  Scenario: 検索条件をクリアする
    Given タグを検索している状態で
    When 検索ボックスを空にすると
    Then 全てのタグが人気順で表示される

  Scenario: タグを選択する
    Given 選択されていないタグが存在する
    When 選択されていないタグが押されると
    Then 該当のタグが選択される

  Scenario: タグの選択を解除する
    Given 選択されているタグが存在する
    When 選択されているタグが押されると
    Then 該当のタグが未選択状態になる

  Scenario: タグを保存する
    When 保存ボタンが押されると
    Then 選択されているタグが保存される

  Scenario: タグ未選択で保存しようとする
    Given タグを何も選択していないとき
    When 保存ボタンが押されると
    Then タグを選択するように、という旨のメッセージが表示され、画面遷移が出来ない
```
